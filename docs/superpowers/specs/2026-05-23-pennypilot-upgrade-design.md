# PennyPilot 全面升级设计文档

**日期:** 2026-05-23
**作者:** SanZ1
**状态:** 已批准

---

## 1. 目标

将 PennyPilot 从一个只跟踪 3 个指数 + 3 只基金的终端工具，升级为覆盖 7 个指数 + 6 只基金、具备完整前端仪表盘的大学生/职场新人理财学习平台。

**核心原则：**
- 零构建、零依赖，可直接打开 `web/index.html`
- Python 后端生成 JSON，前端读取渲染
- 保留终端输出功能
- 深浅主题切换

## 2. 目标用户

大学生和刚工作的年轻人，没有投资经验，需要：
- 低门槛理解市场状态
- 清晰的定投建议和配比
- 交互式学习理财思维
- 复利收益可视化

## 3. 架构

```
Python 后端                          前端
┌─────────────────┐              ┌─────────────────┐
│ data_fetcher.py │──┐           │  index.html     │
│ (akshare/bao)   │  │           │  app.js         │
├─────────────────┤  │  生成     │  styles.css     │
│ valuation.py    │──┼──────────→│                 │
│ (估值判断)      │  │  JSON     │  读取 data.json │
├─────────────────┤  │           │  渲染仪表盘     │
│ catalog.py      │──┘           │  深浅主题切换   │
│ (理财选项库)    │              │  定投模拟器     │
└─────────────────┘              └─────────────────┘
```

- Python 运行后生成 `web/data.json`
- 前端读取 JSON 渲染完整仪表盘
- 无后端服务器，可部署到 GitHub Pages
- 终端输出（`display.py`）保持不变

### 4.6 main.py 流程更新

```
1. 获取指数行情 (7个)
2. 获取估值数据 (6个，恒生科技无PE估值)
```

## 5. 前端设计

### 5.1 整体结构

单页仪表盘，顶部导航锚点跳转到 6 个区域：

1. **市场概览** — 7 个指数的实时行情卡片
2. **估值分析** — PE 分位可视化（绿/黄/红三色 + 进度条）
3. **定投推荐** — 5 只基金的配比条形图 + 实时净值卡片
4. **定投模拟器** — 复利计算器 + SVG 资产增长曲线
5. **理财学习** — 思维训练卡片
6. **检查清单** — 投资前自检

### 5.2 视觉风格

浅色现代 SaaS 风格（主选），支持深色主题切换：
- 主色：#0284c7（蓝）
- 成功：#22c55e（绿）
- 危险：#ef4444（红）
- 警告：#eab308（黄）
- 背景：#f8fafc / 卡片：#fff / 边框：#e2e8f0
- 字体：系统字体栈（Segoe UI, Microsoft YaHei, Arial）

### 5.3 主题系统

CSS 变量实现深浅切换：

```css
:root {
  --bg: #f8fafc;
  --card: #fff;
  --text: #1e293b;
  --text-secondary: #64748b;
  --border: #e2e8f0;
  --accent: #0284c7;
  --success: #22c55e;
  --danger: #ef4444;
  --warning: #eab308;
}

[data-theme="dark"] {
  --bg: #0f172a;
  --card: #1e293b;
  --text: #e2e8f0;
  --text-secondary: #94a3b8;
  --border: #334155;
  --accent: #38bdf8;
  --success: #4ade80;
  --danger: #f87171;
  --warning: #facc15;
}
```

`localStorage` 记住用户偏好，默认跟随系统 `prefers-color-scheme`。

### 5.4 定投模拟器

- 用户输入：每月金额、年数、预期年化
- 计算公式：`FV = PMT × ((1+r)^n - 1) / r`（期末年金）
- SVG 绘制双曲线：投入本金（蓝色）vs 预估市值（绿色）
- 实时更新，输入变化立即重算
- 底部免责提示

### 5.5 估值热力图

用颜色编码展示各指数估值状态：
- 绿色（低估，<30%）：建议买入
- 黄色（合理，30%-70%）：持有
- 红色（高估，>70%）：观望

每个卡片包含：指数名、PE 分位数字、估值状态、信号、进度条。

### 5.6 数据加载策略

1. 优先读取 `data.json`（有最新数据）
2. 若缺失，使用 `app.js` 内嵌的默认数据
3. 显示提示："运行 python -m finance_monitor.main 获取最新数据"
4. 每个区域独立渲染，单个数据缺失不影响其他区域
