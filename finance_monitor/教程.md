## 第一章：认识这个项目

### 这个项目能做什么？

简单说，它是一个**自动化的市场观察助手**：

1. 每天早上自动抓取 A 股三大指数（沪深300、中证500、上证50）的最新行情
2. 获取这些指数的**市盈率（PE）和历史百分位**
3. 根据百分位判断市场是**低估**还是**高估**
4. 给你一个明确的建议：**买入**、**持有**、还是**观望**
5. 把所有信息用漂亮的表格打印在终端里

### 它不能做什么？

- 不能帮你自动下单买卖（需要你自己去支付宝/天天基金操作）
- 不能预测股市涨跌（没有人能）
- 不能保证你赚钱（投资有风险）

### 项目用到了哪些技术？

| 技术 | 作用 | 你需不需要懂？ |
|------|------|---------------|
| Python | 编程语言 | 需要安装，但不用会写 |
| BaoStock | 免费的 A 股行情数据接口 | 不用懂，脚本自动调用 |
| akshare | 免费的金融数据接口 | 不用懂，脚本自动调用 |
| rich | 终端美化输出库 | 不用懂，让表格好看 |

## 第二章：安装 Python

### 2.1 检查是否已安装 Python

按 `Win + R`，输入 `cmd`，回车，打开命令提示符。输入：

```bash
python --version
```

如果显示类似 `Python 3.10.2` 的版本号，说明已安装，跳到 [第三章](#第三章安装项目依赖)。

如果显示"不是内部或外部命令"，说明没安装，继续往下看。

### 2.2 下载 Python

1. 打开浏览器，访问 Python 官网：https://www.python.org/downloads/
2. 点击黄色的 **"Download Python 3.x.x"** 按钮，下载最新版本
3. 下载完成后，双击安装包

### 2.3 安装 Python（关键步骤！）

**这一步非常重要，请仔细看：**

1. 打开安装包后，**先勾选底部的 "Add Python to PATH"**（这一步很多人漏掉！）
2. 然后点 **"Install Now"**
3. 等待安装完成
4. 看到 "Setup was successful" 就装好了

```
┌─────────────────────────────────────────┐
│  ☑ Install launcher for all users       │
│  ☑ Add Python 3.x to PATH  ← 一定要勾！│
│                                         │
│  [Customize installation]  [Install Now] │
└─────────────────────────────────────────┘
```

### 2.4 验证安装

**重新**打开一个 cmd 窗口（之前的窗口不会识别新装的 Python），输入：

```bash
python --version
```

显示版本号就说明安装成功了。

同时验证 pip（Python 的包管理工具）：

```bash
pip --version
```

也显示版本号就 OK。

## 第三章：安装项目依赖

### 3.1 打开终端，进入项目目录

按 `Win + R`，输入 `cmd`，回车。然后输入：

```bash
cd "C:\Users\Xujusheng\Desktop\新建文件夹 (2)\finance_monitor"
```

> **解释：** `cd` 是"进入某个文件夹"的意思。你的项目文件在这个文件夹里。

### 3.2 安装依赖库

输入以下命令，一次安装所有需要的库：

```bash
pip install -r requirements.txt
```

你会看到类似这样的输出：

```
Collecting baostock
  Downloading baostock-0.9.1-py3-none-any.whl
Collecting akshare
  Downloading akshare-1.18.60-py3-none-any.whl
Collecting rich
  Downloading rich-15.0.0-py3-none-any.whl
...
Successfully installed baostock-0.9.1 akshare-1.18.60 rich-15.0.0
```

看到 `Successfully installed` 就说明安装成功了。

### 3.3 如果 pip 很慢怎么办？

国内网络下载 Python 库可能比较慢，可以用国内镜像源加速：

```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

> **解释：** `-i` 参数指定从清华大学的镜像服务器下载，速度会快很多。

## 第五章：看懂输出报告

### 5.1 核心指数行情

这张表告诉你今天三大指数的表现：

| 列名 | 含义 | 怎么看 |
|------|------|--------|
| 指数 | 指数名称 | 沪深300 代表 A 股最大的 300 家公司 |
| 最新价 | 今天的收盘点位 | 数字本身没意义，看涨跌幅 |
| 涨跌幅 | 相比昨天涨了还是跌了 | 绿色=涨，红色=跌 |
| 日期 | 数据日期 | 周末/节假日会显示上一个交易日 |

### 5.2 指数估值分析（最重要！）

这张表是整个报告的核心：

| 列名 | 含义 | 怎么看 |
|------|------|--------|
| PE | 市盈率，衡量股票贵不贵的指标 | 数字越小越便宜 |
| PE百分位 | 当前 PE 在历史中的位置 | 核心判断依据 |
| 估值水平 | 根据百分位自动判断 | 低估/合理/高估 |
| 操作建议 | 脚本给你的建议 | 买入/持有/观望 |

**PE 百分位怎么看？**

```
  0%                    30%         70%                 100%
  ├──────────────────────┼───────────┼──────────────────────┤
  │      低估区间        │  合理区间  │      高估区间        │
  │   建议：大胆买入     │ 建议：持有 │   建议：观望/止盈    │
  │   颜色：绿色         │ 颜色：黄色 │   颜色：红色         │
```

- **PE 百分位 < 30%**：市场便宜，历史上 70% 的时间比现在贵，适合买入
- **PE 百分位 30%-70%**：市场正常，不贵不便宜，按计划定投
- **PE 百分位 > 70%**：市场偏贵，历史上只有 30% 的时间比现在贵，谨慎

### 5.3 基金净值

告诉你关注的基金今天值多少钱，以及相比昨天涨跌了多少。

### 5.4 本月投资计划

根据市场估值自动计算的分配方案：

- **低估时**：160 元投基金 + 40 元存余额宝
- **高估时**：200 元全部存余额宝，暂停定投

### 5.5 综合操作建议

把三个指数的 PE 百分位取平均，给出一个整体判断。这是你做决策的主要参考。

## 第六章：理解代码结构

> 这一章不是必须看的，但如果你想修改脚本，了解结构会有帮助。

### 6.1 文件一览

```
finance_monitor/
│
├── main.py              ← 主入口，运行这个文件
│   作用：串联所有模块，控制执行流程
│
├── config.py            ← 配置文件
│   作用：存放指数列表、基金列表、阈值、预算等参数
│   修改这里可以自定义你的监控目标
│
├── data_fetcher.py      ← 数据抓取
│   作用：调用 BaoStock 和 akshare 获取行情和估值数据
│   你一般不需要改这个文件
│
├── valuation.py         ← 估值分析
│   作用：根据 PE 百分位判断低估/合理/高估
│   作用：计算每月投资分配方案
│   作用：保存历史数据到 history.json
│
├── display.py           ← 终端美化
│   作用：用 rich 库把数据打印成漂亮的表格
│   你一般不需要改这个文件
│
├── requirements.txt     ← 依赖清单
│   作用：告诉 pip 需要安装哪些库
│
├── run_daily.bat        ← 每日运行脚本
│   作用：给 Windows 任务计划程序调用
│
├── setup_task.bat       ← 一键配置定时任务
│   作用：自动在 Windows 任务计划程序中创建定时任务
│
├── history.json         ← 历史数据（自动生成）
│   作用：记录每天的 PE、百分位、收盘价
│   可以用来做长期分析
│
└── 教程.md              ← 你正在看的这个文件
```

### 6.2 数据流向

```
BaoStock API  ──→  data_fetcher.py  ──→  main.py  ──→  display.py  ──→  终端输出
                       ↑                    │
akshare API   ──→  data_fetcher.py         │
                                           ▼
                                    valuation.py  ──→  history.json
```

1. `data_fetcher.py` 从网上抓取数据
2. `main.py` 把数据传给 `valuation.py` 做分析
3. `main.py` 把分析结果传给 `display.py` 打印
4. `valuation.py` 同时把数据保存到 `history.json`

### 6.3 config.py 详解

```python
# 监控的指数列表
INDICES = [
    {"name": "沪深300", "code": "sh.000300", "lg_name": "沪深300"},
    #  name: 显示名称（随便改）
    #  code: BaoStock 的指数代码（不要改）
    #  lg_name: 乐咕乐股的名称（不要改，改了估值就取不到了）
]

# 监控的基金列表
FUNDS = [
    {"name": "天弘沪深300ETF联接A", "code": "000961"},
    #  name: 显示名称（随便改）
    #  code: 基金代码（去天天基金网查）
]

# 估值阈值
UNDERVALUED_THRESHOLD = 30   # 低于 30% = 低估
OVERVALUED_THRESHOLD = 70    # 高于 70% = 高估

# 每月预算
MONTHLY_BUDGET = 200.0       # 每月 200 元
FUND_RATIO = 0.8             # 80% 投基金
SAVINGS_RATIO = 0.2          # 20% 存余额宝
```

## 第七章：自定义配置

### 7.1 修改每月预算

打开 `config.py`，找到：

```python
MONTHLY_BUDGET = 200.0
```

改成你的实际预算，比如 150 元：

```python
MONTHLY_BUDGET = 150.0
```

### 7.2 修改投资比例

默认 80% 投基金、20% 存余额宝。如果你想更保守（多存点应急钱）：

```python
FUND_RATIO = 0.6      # 60% 投基金
SAVINGS_RATIO = 0.4    # 40% 存余额宝
```

### 7.3 添加或更换基金

去天天基金网（https://fund.eastmoney.com/）搜索你想要的基金，找到基金代码。

比如你想加一个"易方达中证红利ETF联接A"，代码是 009052：

```python
FUNDS = [
    {"name": "天弘沪深300ETF联接A", "code": "000961"},
    {"name": "天弘中证500ETF联接A", "code": "000962"},
    {"name": "华夏上证50ETF联接A", "code": "001051"},
    {"name": "易方达中证红利ETF联接A", "code": "009052"},  # 新加的
]
```

### 7.4 修改估值阈值

默认 30% 以下算低估，70% 以上算高估。如果你想更保守（只在特别便宜时才买）：

```python
UNDERVALUED_THRESHOLD = 20   # 低于 20% 才算低估
OVERVALUED_THRESHOLD = 80    # 高于 80% 才算高估
```

### 7.5 修改后怎么生效？

保存文件，下次运行 `python main.py` 就会自动用新配置。

## 第八章：设置每天自动运行

### 方法一：一键脚本（推荐）

1. 打开文件夹 `finance_monitor`
2. 找到 `setup_task.bat`
3. **右键** → **以管理员身份运行**
4. 看到"定时任务创建成功"就搞定了

### 方法二：手动配置

如果你不想用脚本，可以手动设置：

1. 按 `Win + R`，输入 `taskschd.msc`，回车
2. 打开了"任务计划程序"窗口
3. 右侧点击 **"创建基本任务"**

4. **第一步：名称和描述**
   - 名称：`理财监控`
   - 描述：`每天早上 8:30 自动运行市场监控`
   - 点"下一步"

5. **第二步：触发器**
   - 选择"每天"
   - 点"下一步"
   - 开始日期：选今天的日期
   - 时间：改成 `08:30:00`
   - 每隔：`1` 天
   - 点"下一步"

6. **第三步：操作**
   - 选择"启动程序"
   - 点"下一步"
   - 程序或脚本：点"浏览"，找到 `finance_monitor\run_daily.bat`
   - 起始于（可选）：填 `C:\Users\Xujusheng\Desktop\新建文件夹 (2)\finance_monitor`
   - 点"下一步"

7. **第四步：完成**
   - 勾选"当单击完成时，在打开的任务属性对话框中进行高级设置"（可选）
   - 点"完成"

8. **高级设置（可选）**
   - 在弹出的属性窗口中：
   - 勾选"如果任务失败，按以下频率重新启动"
   - 设置"如果任务运行时间超过：30 分钟"
   - 确定

### 方法三：只用 bat 文件手动运行

如果你不想设置自动运行，也可以每次手动双击 `run_daily.bat` 来运行。

### 管理定时任务

设置好之后，你可能想查看、修改或删除定时任务：

| 操作 | 步骤 |
|------|------|
| 查看任务 | 打开任务计划程序 → 左侧"任务计划程序库" → 找到"理财监控" |
| 手动运行 | 右键任务 → "运行" |
| 暂停任务 | 右键任务 → "禁用" |
| 删除任务 | 右键任务 → "删除" |

或者用命令行：

```bash
# 查看任务
schtasks /query /tn "ValuEye"

# 手动运行
schtasks /run /tn "ValuEye"

# 删除任务
schtasks /delete /tn "ValuEye" /f
```

## 第九章：常见问题排查

### Q1：运行时报 `python` 不是内部命令

**原因：** Python 没有安装，或者没有添加到系统 PATH。

**解决：**
1. 重新安装 Python，记得勾选 "Add Python to PATH"
2. 或者手动添加 PATH：
   - 右键"此电脑" → 属性 → 高级系统设置 → 环境变量
   - 在"系统变量"中找到 `Path`，编辑，添加 Python 的安装路径

### Q2：pip install 很慢或超时

**原因：** 国内网络访问 Python 官方源较慢。

**解决：** 使用清华镜像源：

```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q3：某个指数/基金数据获取失败

**原因：** 可能是网络波动，或者非交易时间数据未更新。

**解决：**
- 交易日（周一到周五）运行效果最好
- 周末和节假日，数据会显示上一个交易日的
- 单个指数失败不影响其他指数，脚本会继续运行

### Q4：估值数据全部获取失败

**原因：** akshare 版本太旧，或者数据源接口变化。

**解决：**

```bash
pip install --upgrade akshare
```

### Q5：中文显示乱码

**原因：** Windows 终端默认编码不是 UTF-8。

**解决：**
1. 运行前先执行 `chcp 65001`
2. 或者用 `run_daily.bat` 运行（里面已经设置了编码）
3. 或者在 Windows 设置中：时间和语言 → 管理语言设置 → 更改系统区域设置 → 勾选"Beta 版: 使用 Unicode UTF-8 提供全球语言支持"

### Q6：周末运行显示的数据不是最新的

**这是正常的。** 股市周末和节假日不开盘，所以数据会是上一个交易日的。脚本会自动取最近的交易日数据。

### Q7：我想看看历史数据在哪里

历史数据保存在 `history.json` 文件中。你可以用记事本打开查看，格式如下：

```json
{
  "沪深300": [
    {"date": "2026-05-14", "pe": 14.17, "pe_percentile": 22.0, "close": 4914.6},
    {"date": "2026-05-13", "pe": 14.33, "pe_percentile": 22.7, "close": 4998.34}
  ]
```

## 附录：理财基础知识

### 什么是指数基金？

- **指数**：一篮子股票的集合。沪深300 就是 A 股最大的 300 家公司的股票打包在一起。
- **指数基金**：按照指数的成分，买入同样比例的股票。你买一份指数基金，相当于同时买了 300 家公司的股票。
- **ETF 联接基金**：可以在支付宝、天天基金等平台直接买的指数基金。

### 什么是定投？

- **定投**：每月固定日期，投入固定金额，不管市场涨跌都买。
- **好处**：不用择时，避免追涨杀跌，长期下来成本会被摊平。
- **纪律**：定投的核心是"坚持"，不是"聪明"。

### 什么是 PE 和百分位？

- **PE（市盈率）**：股价 ÷ 每股收益。PE 越低，股票越"便宜"。
- **PE 百分位**：当前 PE 在过去几年中的位置。22% 意味着历史上 78% 的时间 PE 比现在高。

### 为什么 200 元也要理财？

1. **培养习惯**：理财是一种习惯，金额不重要，纪律才重要
2. **复利效应**：即使每月 200 元，坚持 10 年，按年化 8% 计算，也能积累约 3.6 万元
3. **学习成本低**：200 元的"学费"比以后 2 万元的"学费"便宜多了

### 每月操作 SOP（标准作业程序）

```
每月 1 号（发生活费后）
    │
    ├── 40 元 → 转入余额宝（应急资金）
    │
    └── 160 元 → 根据脚本建议操作
         │
         ├── 脚本显示"低估/买入" → 分 3 份买入基金
         │
         ├── 脚本显示"合理/持有" → 分 3 份买入基金
         │
         └── 脚本显示"高估/观望" → 160 元也转入余额宝
```
